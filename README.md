# Banco de Dados II | Consultas e Extração em SQL

Este repositório reúne os scripts práticos desenvolvidos para solidificar o domínio da linguagem SQL, com foco exclusivo na extração e análise avançada de dados. O objetivo é construir a base técnica necessária para a formulação de consultas complexas, essenciais na integração com sistemas de backend.

## O que foi desenvolvido?

O escopo atual foca na exploração analítica de bancos relacionais. Os códigos demonstram o cruzamento de múltiplas entidades e a aplicação de lógicas de filtragem para simular a extração real de informações de negócio.

## Conceitos Aplicados

* **Consultas Relacionais Avançadas:** Uso aprofundado de junções (JOINs) para a unificação de dados distribuídos em diversas tabelas.
* **Agrupamento Analítico e Sumarização:** Aplicação de funções de agregação (SUM, AVG, COUNT, MAX, MIN) para gerar métricas e relatórios estruturados.
* **Subconsultas (Subqueries) e Filtros Complexos:** Construção de consultas aninhadas e filtros lógicos para a extração refinada de conjuntos de dados.
